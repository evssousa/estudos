Uma associação é um relacionamento entre classes que permite que uma instância de uma classe faça a outra realizar uma ação.

É como dizer para a aplicação que um objeto de um tipo está conectado, ou depende, de um objeto de outro tipo.

Exemplos:

- Objetos de uma classe Pessoa possuem um atributo que é um objeto da classe Endereco.
- Objetos de uma classe Produto possuem um atributo que é um array de objetos de classe Categoria.
